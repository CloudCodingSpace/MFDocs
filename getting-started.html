Getting Started


Prerequisites


First and foremost, you need to have some software already installed, in order for MeltedForge to work.

  1. Git.
  2. A 64-bit desktop OS.
  3. Vulkan SDK
  4. A GPU with the latest drivers which supports Vulkan 1.2 atleast.
  5. CMake.
  6. A modern C & C++ compiler installed. (Preferably MSVC on windows, and GCC/G++ with the MinGW toolchain)

Downloading the codebase

Make sure to clone the repository from github. Use this command in the command line to clone it :- 

git clone --recursive https://github.com/CloudCodingSpace/MeltedForge "name of the workspace"
